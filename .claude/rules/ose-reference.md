# OSE Reference

When clarifying classic D&D rules (attack matrices, turn undead, morale, saving throws,
monster stats, spell lists, XP tables, etc.), consult the OSE SRD Wiki at:

  `reference/OSE.SRD.Wiki-main/`

Organized as:
- `1. Characters/` — stats, saves, encumbrance
- `2. Classes/` — Fighter, Cleric, Magic-User, Elf, Dwarf, Halfling progressions
- `3. Equipment & Services/`
- `4. Magic/` — spells
- `5. Adventures/` — dungeon procedures, time, movement
- `6. Running Adventures/` — encounters, morale, reaction
- `7. Monsters/` — all monster stats
- `8. Treasures/`

**Override rule:** The user will explicitly state when Venturekeep intentionally deviates
from OSE rules (e.g., "all HD are d6" overrides OSE class-specific hit dice). Do not
"correct" these overrides — apply the stated rule instead.
